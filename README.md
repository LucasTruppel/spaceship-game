# spaceship-game
Projeto da disciplina Sistemas Operacionais 1, com intuito de criar uma biblioteca de threads em nível de usuário e usá-la em um jogo de naves.

## Pré-requisitos

- Linguagem c++
- Compilador g++
- Ferramenta make

## Compilar

```shell
make
```

## Executar

```shell
./main
```

## Limpar os binários

```shell
make clean
```
