# threads-library
Projeto da disciplina Sistemas Operacionais 1, com intuito de criar uma biblioteca de threds em nível de usuário.

## Pré-requisitos

- Linguagem c++
- Compilador g++
- Ferramenta make

## Compilar

```shell
make all
```

## Executar

```shell
./program
```

## Limpar os binários

```shell
make clean
```
